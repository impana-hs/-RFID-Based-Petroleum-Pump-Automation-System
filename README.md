<h2>📌 Project Overview</h2>

<p>
The <b>RFID Based Petroleum Pump Automation System</b> is an embedded automation project developed using <b>ATmega328 Microcontroller</b>, <b>RFID Technology</b>, <b>LCD Display</b>, <b>Keypad</b>, and <b>Relay Module</b>. The system enables secure, cashless, and automated fuel dispensing by authenticating users through RFID cards and controlling the fuel pump automatically based on user input.
</p>

<h2>🚀 Features</h2>

<ul>
  <li>RFID-based user authentication.</li>
  <li>Secure and cashless fuel transactions.</li>
  <li>Fuel quantity selection through keypad.</li>
  <li>Real-time balance verification.</li>
  <li>Automatic pump motor control using relay module.</li>
  <li>LCD-based user interaction and transaction display.</li>
  <li>Unauthorized access prevention.</li>
  <li>Real-time transaction monitoring.</li>
  <li>Improved efficiency with reduced manual intervention.</li>
</ul>

<h2>⚙️ System Workflow</h2>

<ol>
  <li>User scans RFID card.</li>
  <li>System validates card details.</li>
  <li>Available balance is displayed on LCD.</li>
  <li>User enters required fuel quantity using keypad.</li>
  <li>System verifies sufficient balance.</li>
  <li>Relay activates the fuel pump.</li>
  <li>Fuel dispensing process begins.</li>
  <li>Transaction details are updated.</li>
  <li>Pump automatically stops after completion.</li>
</ol>
